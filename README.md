## project05_DateGuide
> chat GPT API 활용한 채팅 웹 서비스
<br/>

### 프로젝트의 목적
> 1. 데이트 코스가 고민될 때 지역과 데이트 컨셉만 입력하면 알아서 데이트 코스를 추천하여 사용자의 고민을 해결할 수 있습니다. <br/>
> 2. 채팅 형식의 스타일로 데이트 코스에 대한 추가적인 질문(ex. 교통, 위치, 입장료 등)에 대한 답변도 편리하게 받을 수 있습니다.
